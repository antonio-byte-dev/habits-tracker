# Habit Tracking App
The objective of this app is to let you create multiple habits and log the time spent on each one.
## Already Done
As of right now it's possible to create habits, delete them and update them. Each habit is stored in a mongoDB(stored in a docker container). This works by creating a user using firebase auth. Backend and Frontend are in different containers.
